# L17
Symmetric matrices
## 17.1 On dot product for complex vectors
> ##### Def 1: Conjugate transpose 共轭转置
> 又称 Hermitian transpose
$$A^*=(\overline{A})^T=\overline{(A^T)}$$
>
>又记做 $A^H,A^\dagger$

例如，对于矩阵 $A=\begin{bmatrix}
1 & i \\
0 & 2
\end{bmatrix},A^T=\begin{bmatrix}
1 & 0 \\
i & 2
\end{bmatrix},A^*=\begin{bmatrix}
1 & 0 \\
-i & 2
\end{bmatrix}$

> ##### Def 2: Dot product, norm, and orthogonality for complex vectors 复数向量的点乘、范数以及正交性
> 1. 对于实数向量 $x,y\in\R^n$
> dot product: $x\cdot y=x^Ty=\sum x_iy_i$
norm: $||x||=\sqrt{x\cdot x}=\sqrt{\sum x_i^2}$

> ##### Def 3: Symmetric matrices
> 对称矩阵 $A\in\R^{n\times n}$ 具有如下性质
> 1. 特征根均为实数
> 2. diagonalizable 可对角化的，即其 Jordan form 为对角矩阵
> 3. 对应不同特征根的特征向量互相正交

## 18.1 正定矩阵

> **Def 1: Positive definite matrices (eigenvalues base)**
A symmetric matrix $A\in\Complex^{n\times n}$ is said to be positive definite if all its **eigenvalues are positive**

> **Def 2: Energy base definition**
$x^TAx$ is interpreted as "energy of $x$", 正定就意味着对于任意 $x\in\R^n,x\neq 0$, 都有 $x^TAx > 0$ 

**Theorem 1**: Def 1 与 Def 2 可以相互转化
- Proof (1) Def 1 $\to$ Def 2
- Proof (2) Def 2 $\to$ Def 1

**等效性质** (以下性质可于正定互推)
- All pivots are positive
- Upper left deteminants are positive ???
- $A=B^TB, B\in\R^{m\times x}$ with independent columns

## 18.2
> **Def 1: Negative definite**
> 1. Eignvalues of $A$ are negative
> 2. $x^TAx<0$ if $x\neq0$ 

> **Def 2: Positive semi-definite**
> 1. Eignvalues of $A$ are non-negative
> 2. $x^TAx \geq 0$ if $x\neq 0$ 
>
> 对于 Negative semi-definite，性质恰好相反

> **Def 3: Indefinite**
> $A$ is indefinite 如果同时存在正的和负的特征值



  private void RunScript(object x, object y, ref object A)
  {
    List<Point3d> centers = new List<Point3d>();
    List<List<Vector3d>> vecs = new List<List<Vector3d>>();

    // 储存不同类球节点的 index
    List<int> type = new List<int>();

    Point3d center;
    List<Vector3d> vec;

    Point3d tcenter;
    List<Vector3d> tvec;
    Boolean ifadd = true;

    // 遍历所有的点
    for (int i = 0; i < centers.Count; i++)
    {
      center = centers[i];
      vec = vecs[i];
      Vector3d sumVec = new Vector3d(0, 0, 0);
      for (int k = 0; k < vec.Count; k++)
        sumVec += vec[k];


      // 遍历所有的已有类别
      for (int j = 0; j < type.Count; j++)
      {
        tcenter = centers[type[j]];
        tvec = vecs[type[j]];
        // 首先判断向量个数是否一致
        if (vec.Count == tvec.Count)
        {
          Vector3d sumTVec = new Vector3d(0, 0, 0);
          for (int k = 0; k < tvec.Count; k++)
            sumTVec += tvec[k];
          // 先判判断各条边的合成线是否一致
          if ((sumTVec - sumVec).Length < 0.00001)
          {
            // 最后判断到底是否一致
            /// 可以用于排除对称的情况               
            if (CompareNodes(vec, tvec, sumVec, sumTVec))
            {
              ifadd = false;
            }
          }
        }

        if (ifadd == true)
        {
          type.Add(i);
        }
      }
            

    A = type;
  }

  private Boolean CompareNodes(List<Vector3d> vecs1, List<Vector3d> vecs2, Vector3d mid1, Vector3d mid2)
  {
    // 计算夹角
    double angle = Vector3d.VectorAngle(mid1, mid2);
    Vector3d axis = new Plane(new Point3d(0, 0, 0), mid1, mid2).Normal;
    // 尝试转动 mid2 以求转动方向
    mid2.Rotate(angle, axis);
    if ((mid1 - mid2).Length > 0.00001)
      angle = -angle;

    // 转动 vecs2
    for (int i = 0; i < vecs2.Count; i++)
    {
      vecs2[i].Rotate(angle, axis);
    }

    // 计算转动后 vecs2 中的每个向量与 vecs1 中的每个向量的夹角
    List<List<double>> angleL = new List<List<double>>();

    for (int i = 0; i < vecs2.Count; i++)
    {
      List<double> anglel = new List<double>();
      for (int j = 0; j < vecs1.Count; j++)
        anglel.Add(Vector3d.VectorAngle(vecs2[i], vecs1[j]));
      angleL.Add(anglel);
    }

    // 如果 vecs1 与 vecs2 的 type 相同，那么 angleL 中的所有 anglel 中必定有一个数字是一样的
    List<double> anglesA = angleL[0];
    List<double> anglesB;
    for (int i = 0; i < angleL.Count - 1; i++)
    {
      anglesB = angleL[i + 1];
      List<double> same = new List<double>();
      for (int j = 0; j < anglesA.Count; j++)
      {
        for (int k = 0; k < anglesB.Count; k++)
        {
          if ((Math.Abs(anglesA[j]) - Math.Abs(anglesB[k])) < 0.000001)
            same.Add(anglesB[k]);
        }
      }
      if (same.Count == 0)
        break;

      anglesA = same;
    }

    if (anglesA.Count != 0)
      return true;

    return false;
  }
