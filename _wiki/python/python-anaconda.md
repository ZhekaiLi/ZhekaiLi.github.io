Using Command Line to mange python environments & use Anaconda

打开 jupyter
```py
jupyter lab
jupyter notebook
```

关闭 jupter `Crtl + C `

管理环境
```py
conda env list # 查看所有虚拟环境
conda list     # 查看当前环境下安装的包

conda 
```
